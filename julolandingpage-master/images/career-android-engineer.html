Android Engineer

Responsibilities

  • You will be responsible for core development of JULO’s Android app
  • You will develop new, awesome features to be used by tens of thousands of JULO app users
  • Bring expertise of Android mobile development to the team
  • Write efficient, maintainable code for mobile phones
  • Advocate best practices in Android programming, including clean code and clear design pattern
  • Own release and QA cycles of your and others’ codes
  • You will develop quick prototyping for immediate assessment with product team
  • Perform pair code review with other Android developers
  • Work with customer support to fix reported bugs

Requirements

  • Exceptional CAN-DO ATTITUDE
  • Min 1yrs work experience as Native Android app developer
  • Proficient with Android Studio and Java
  • Experience in leading an engineering team
  • Ability to manage projects and people
  • Experience with XML, JQuery mobile, SQLite, Butter Knife / Dagger, Retrofit, RxJava
  • Experience with backend integration via RESTful APIs
  • Experience with Android automated UI testing
  • Understand and has experience with MVC/MVP design pattern
  • Writes / delivers clean, organized codes that can be easily understood by other team members
  • Must have excellent managerial and supervisory skills, and verbal and written communication skills.(Bahasa Indonesia and English)
  • Strong in troubleshooting
  • Bachelor’s degree in Computer Science/ Engineering/ IT, with 1 years of software development experience

Nice to have experience

Familiar with deployment tools, e.g. CI/CD CircleCI, Ansible, Chef

Familiar with Github, cloud computing (AWS, Azure, etc), UNIX OS

Master’s degree preferred
